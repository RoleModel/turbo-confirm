# Change Log

## [v2.2.0] Nov 12, 2025

- Add `inert` attribute management on Show & Hide.

## [v2.1.1] Oct 31, 2024

- Re-Add support for old versions of Turbo.

## [v2.1.0] Oct 31, 2024

- drop support for old versions of Turbo.

## [v2.0.2] Apr 26, 2024

- `link_to` actually works now, for real this time.

## [v2.0.1] Apr 10, 2024

- Add local development scripts.
- Resolve a Webpack compile bug.

## [v2.0.0] Apr 2, 2024

- Simplified integration setup. (Breaking Change)
- Native HTML Dialog Element support.
- Direct invocation Support: `turboConfirm.confirm('Are you sure?')`.
- New Public Method: `confirmWithContent`.
- Helpful error messages.

## [v1.1.2] Sep 30, 2023

- Properly handle `cancel` events.

## [v1.1.1] Sep 27, 2023

- Improved README.md documentation.
- Fix a `link_to` bug.

## [v1.1.0] Aug 29, 2023

- Official `link_to` support.

## [v1.0.0] Apr 10, 2023

- Custom ContentSlot support.
- Add `data-confirm-button` default ContentSlot.
